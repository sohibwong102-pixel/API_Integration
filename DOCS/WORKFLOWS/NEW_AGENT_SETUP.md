# 🤖 NEW_AGENT_SETUP PLAYBOOK 😎🔥

> cara bikin agent baru tanpa bikin ecosystem jadi perang dunia 😭🔥

---

# 🧠 Tujuan

Supaya:
- role agent jelas
- responsibility tidak tabrakan
- orchestration tetap civilized 😎🔥

---

# 🏷️ Naming Rule

Gunakan format:

```txt
ROLE_SPECIALIZATION
```

Contoh:
- BACKEND_EXECUTOR
- PROMPT_OPTIMIZER
- REVIEW_SUMMARIZER

---

# 🔥 Agent Harus Punya

```txt
[ ] role jelas
[ ] boundary jelas
[ ] input jelas
[ ] output jelas
[ ] tidak overlap brutal 😭
```

---

# ⚙️ Minimal Structure

Agent wajib punya:
- ROLE
- MISSION
- RULES
- OUTPUT FORMAT
- RESTRICTIONS

---

# ❌ Jangan Dilakukan

## Agent terlalu general

Contoh buruk:

```txt
AI SUPER ULTRA EVERYTHING AGENT 😭🔥
```

Hasil:
```txt
chaos
```

---

## Agent pegang semua logic

Karena:
```txt
single goblin point of failure 😭🔥
```

---

# 🧩 Philosophy

```txt
1 agent
1 responsibility 😎🔥
```

Kalau 1 agent:
- coding
- review
- architecture
- planning
- debugging
- deployment

Maka:
```txt
itu bukan agent 😭
itu tumbal 😭🔥
```

---

# 😭 Final Doctrine

```txt
orchestration bagus bukan karena agent pintar

tapi karena role jelas 😎🔥
```
